# SEG-projekt

### How to use:

Keys 1 and 2 switch between:
* 1 - marble mode: left click to create marble 
* 2 - track mode: left click twice, once starting point and once on endpoint to create a track

Hold Middle click to move around <br>
If on touchpad, use WSAD

Hold Right click to delete

Mouse wheel or touchpad to zoom
